# Hide Apps Menu

This Odoo module hides the **Apps** menu from non-admin users by restricting it to the `base.group_system` group.

## Install

1. Copy the `hide_apps_menu` folder into your custom addons path.
2. Restart Odoo.
3. In Odoo, enable Developer Mode.
4. Go to **Apps** → **Update Apps List**.
5. Install **Hide Apps Menu**.

## Result

- Users with **Settings** access can still see **Apps**
- Non-admin users cannot see **Apps**
